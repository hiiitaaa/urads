## ゴール
自分のThreads投稿をアルゴリズムの仕組みに基づいて分析し、
何がウケたか・なぜウケたか・次にどうすべきかを明確にする。

## Threadsアルゴリズム知識（分析の前提）

### 公式ランキングシグナル（Meta Transparency Center）
Threadsは以下5つを予測してランキング:
1. いいね確率
2. リプライクリック確率
3. フォロー確率
4. プロフィールクリック確率
5. スクロール通過抵抗（滞在時間）

### エンゲージメントスコア計算式
```
スコア = いいね×1 + リプライ×5 + リポスト×4 + 引用×3
```
重み付けの根拠:
- リプライ×5: Threads閲覧時間の50%がリプライ上（Meta Japan公式）。自分への返信で42%ブースト
- リポスト×4: 拡散行動（高コスト）
- 引用×3: 議論喚起（コンテンツ追加が必要）
- いいね×1: 最低コスト行動

### ゴールデンアワー
最初の60分のエンゲージメント速度がリーチを決める。
30分で50いいね > 24時間で100いいね。

### 2024年11月の変更
フォロワーへの配信優先。バイラルよりフォロワー基盤構築が重要。

## ワークフロー

### Step 1: データ取得
```bash
curl -s http://localhost:8787/accounts
```
アカウント一覧を取得。最初のアカウントIDを使用。

### Step 2: 投稿履歴取得
```bash
curl -s "http://localhost:8787/posts?status=posted"
```

### Step 3: エンゲージメントスコア計算
各投稿に対して算出:
```
スコア = いいね×1 + リプライ×5 + リポスト×4 + 引用×3
```
上位3件をトップ、下位3件をワーストとして抽出。

### Step 4: アルゴリズム適合度分析
各投稿を以下の軸で評価:

**会話誘発力（最重要）**:
- 問いかけ型か宣言型か？（問いかけ型がリプライを生む）
- リプライチェーンの深さは？
- 自分が返信しているか？（42%ブースト効果）

**コンテンツ品質**:
- 滞在時間を稼げる内容か？（長く読ませる構成）
- 教育的/問題解決/エンタメ価値があるか？
- オリジナルな視点があるか？

**フォーマット効果**:
- テキストのみ / 画像付き / 動画（動画: エンゲージメント率5.55%、テキストの2倍）
- 長さ（短文100字以下/中文100-300字/長文300字以上）

**時間帯効果**:
- 投稿時間とスコアの相関
- ゴールデンアワー（最初の60分）に反応を得られたか？

**アルゴリズムNGチェック**:
- エンゲージメントベイト（「いいねして」等）→ 抑制される
- 同じ内容の連投 → 抑制される
- 他SNSからのコピペ感 → 抑制される

### Step 5: トップ投稿の成功要素
なぜスコアが高いのかを上記の軸で具体的に説明。

### Step 6: ワースト投稿の問題点
なぜスコアが低いのかを上記の軸で具体的に説明。

### Step 7: 読者心理分析
- トップ投稿: 読者はなぜリプライしたくなったか？どんな感情が動いたか？
- ワースト投稿: 読者はなぜスクロール通過したか？何が足りなかったか？

### Step 8: 改善提案（アルゴリズム最適化）
具体的な改善ポイントを3つ以上。アルゴリズムの仕組みに基づいた根拠付き。

### Step 9: スキルコンテキスト保存
```bash
curl -s -X POST http://localhost:8787/chat/skill-context \
  -H "Content-Type: application/json" \
  -d '{"skill_name":"post-analyze","output_summary":{...分析結果...}}'
```
