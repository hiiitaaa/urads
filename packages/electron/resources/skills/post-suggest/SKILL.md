## ゴール
投稿分析 + 競合分析の結果を元に、Threadsアルゴリズムに最適化されたバズる投稿を3案提案する。
60点の叩き台 → 自己評価 → アルゴリズム最適化 → 100点版の完全なループを実行する。

## Threadsアルゴリズム知識（提案の前提）

### エンゲージメントスコア
```
スコア = いいね×1 + リプライ×5 + リポスト×4 + 引用×3
```

### アルゴリズムが重視すること
1. **リプライが最重要** — 閲覧時間の50%がリプライ上。問いかけ型で会話を誘発せよ
2. **ゴールデンアワー** — 最初の60分のエンゲージメント速度がリーチを決める
3. **自分への返信で42%ブースト** — 投稿後すぐに自分でもリプライする前提で設計
4. **滞在時間** — 読み止まる仕掛け（「えっ？」と思わせる冒頭、ストーリー展開）
5. **フォロワー重視** — 2024年11月以降、フォロワーへの配信が優先
6. **テキスト+画像/動画** — テキストのみより2倍のエンゲージメント率

### アルゴリズムNG（必ず避ける）
- エンゲージメントベイト（「いいねして」）→ 即抑制
- 他SNSからのコピペ → 抑制
- 同じ内容の繰り返し → 抑制
- リンクスパム → 抑制（ただし適度なリンクは現在OK）

## 前提条件チェック【必須】
投稿分析と競合分析の結果が必要。なければユーザーに先に実行を促す。
```bash
curl -s "http://localhost:8787/chat/skill-context?skill_name=post-analyze"
curl -s "http://localhost:8787/chat/skill-context?skill_name=research-competitor"
```
どちらかがnullの場合 → 「先に投稿分析/競合分析を実行してください」と伝えて終了。

## ワークフロー

### Step 1: 分析データ収集
- 投稿分析の結果（スコアランキング・成功要素・読者心理・改善ポイント）
- 競合分析の結果（バズパターン・差分・取り入れるべき要素）

### Step 2: トレンド確認
今バズっている投稿をキーワード検索:
```bash
curl -s "http://localhost:8787/research/search?q=占い"
```
（ユーザーのジャンルに合わせたキーワードで検索）

### Step 3: 初回提案（60点の叩き台）
3案を作成。各案に含めるもの:
- 投稿テキスト（500文字以内）
- どのバズ要素を取り入れたか
- ターゲット読者層
- **会話誘発の仕掛け**（問いかけ、意見表明、ツッコミ待ち等）
- **ゴールデンアワー対策**（投稿直後に自分で追加リプライする想定テキスト）

**「ズラシ」の原則**: 競合の丸パクリではなく、自分のアカウントのトーン・世界観に合わせてアレンジする。

### Step 4: 自己評価（各案に対して）
以下の観点で評価:

1. **エンゲージメントスコア予測**: この投稿が得られるいいね・リプライ・リポスト・引用の予測値とスコア
2. **リプライ誘発力**: 読者がリプライしたくなる仕掛けがあるか？（スコアの×5に直結）
3. **読者感情の動き**: 最初に○○と感じ → 次に○○と思い → 最終的に○○したくなる
4. **ゴールデンアワー適性**: 最初の60分で反応を得られる構成か？
5. **滞在時間**: 読み止まる仕掛けがあるか？（スクロール通過抵抗）
6. **アカウント整合性**: 自分の世界観と一致しているか？
7. **Threads最適化チェック**:
   - 500文字以内か？
   - ハッシュタグ1個以内か？
   - ネイティブ感があるか？
   - フォロワーとの関係性を強化する内容か？（2024年11月変更対応）
8. **差別化**: 競合と似すぎていないか？

### Step 5: 改善
評価結果を元に3案を修正。特に:
- リプライ誘発力が弱い案 → 問いかけ要素を追加
- 滞在時間が短そうな案 → 冒頭にフック追加
- ゴールデンアワー対策が不十分 → 自分用リプライテキストを追加
何が問題でどう直したかを明記する。

### Step 6: 最終提案（100点版）
改善後の3案を出力。

### Step 7: スキルコンテキスト保存
```bash
curl -s -X POST http://localhost:8787/chat/skill-context \
  -H "Content-Type: application/json" \
  -d '{"skill_name":"post-suggest","output_summary":{...3案...}}'
```
