# Threads アルゴリズム分析

## 公式情報（Meta Transparency Center）

Threadsのフィードは**3段階パイプライン**で動作:
1. **コンテンツ収集** — 全公開投稿 + フォロー中アカウントの投稿
2. **シグナル分析** — エンゲージメント履歴・興味・行動パターン
3. **ランキング・スコアリング** — 価値予測が高いコンテンツが上位

### 5つの予測ターゲット（公式）

| 予測対象 | 入力シグナル |
|---------|------------|
| **いいね確率** | 過去のいいね、著者とのエンゲージメント履歴、フィード閲覧パターン |
| **リプライクリック確率** | 他ユーザーのリプライクリック数、著者との関係性、直近の活動 |
| **フォロー確率** | Threads活動量、フィード露出、最近のフォロー、Instagram上での著者プロフィール閲覧 |
| **プロフィールクリック確率** | 他ユーザーのクリック数、Instagram上での著者閲覧履歴 |
| **スクロール通過抵抗** | 複数回閲覧したか、著者にいいねしたか、投稿の総閲覧数 |

## エンゲージメントの重み付け（コミュニティ分析）

**行動コストが高い＝重みが大きい:**
1. **滞在時間**（最重要 — 読んで止まったか）
2. **リプライ速度**（最初の60分が勝負）
3. **保存/シェア**（いいねより重い）
4. **フォロー獲得**（究極の品質シグナル）
5. **いいね**（最も軽い行動、最小の重み）

## エンゲージメント速度（ゴールデンアワー）

- **最初の30分で50いいね > 24時間で100いいね**
- 投稿後60分以内のリプライ速度が最も強いブーストシグナル
- 自分のコメントへの返信で**42%**のエンゲージメント向上（全プラットフォーム最高）

## コンテンツフォーマット

- **動画**: エンゲージメント率5.55%（テキストの約2倍）
- **テキスト+画像**: テキストのみ or 画像のみ より高パフォーマンス
- **テキストのみ**: 投稿の63%を占めるが、会話誘発型ならアルゴリズム優遇
- **カルーセル**: 特に日本で伸びている

## ブーストされるコンテンツ

- 会話を生む投稿（問いかけ、議論誘発）
- 教育的・問題解決・エンタメ価値のある内容
- トレンドトピックへのタイムリーな反応
- オリジナルな視点・新しい切り口
- 早期に急速なエンゲージメントを得た投稿
- 一貫して活動するアカウント
- URL/リンク付き投稿（2025年から優遇に変更）

## 抑制されるコンテンツ

- **エンゲージメントベイト**（「いいねしてね」「フォローして」）
- スパム行動（同じ内容の連投、大量投稿）
- バイラルコンテンツの無断転載
- 自動化の乱用（一部のスケジューリングツール）
- 購入フォロワー/エンゲージメント
- 会話を生まない低品質投稿
- ファクトチェッカーにフラグされた内容

## 2024年11月の大きな変更

Adam Mosseriが発表:
- **フォロー中のアカウントのコンテンツを優先**
- 知らないアカウントからのおすすめ（unconnected reach）は減少
- つまり**フォロワー基盤の構築**がバイラルより重要に

## リプライの重要性

- **Threadsの閲覧時間の約50%がリプライ上**（Meta Japan公式）
- Mosseri:「全リプライの価値の合計 ≒ 全投稿の価値の合計」
- リプライはセカンダリではなくファーストクラスのランキングシグナル

## Uradsへの実装方針

### エンゲージメントスコア計算式

閲覧数が取れないため、行動コストの重み付けでスコアリング:

```
スコア = いいね×1 + リプライ×5 + リポスト×4 + 引用×3
```

重み付けの根拠:
- リプライ×5: 最重要シグナル（Mosseri公式発言 + 42%ブースト効果）
- リポスト×4: 拡散行動（高コスト）
- 引用×3: 議論喚起（コンテンツ追加が必要）
- いいね×1: 最低コスト行動

### 投稿分析で見るべき指標
1. **リプライ率** = リプライ数 / フォロワー数
2. **エンゲージメント速度** = 最初のN時間のエンゲージメント
3. **会話深度** = リプライチェーンの長さ
4. **フォーマット効果** = テキストのみ vs 画像付き vs 動画
5. **時間帯効果** = 投稿時間とエンゲージメントの相関

### 投稿提案で重視すべき要素
1. 会話を誘発する構成（問いかけ型 > 宣言型）
2. 最初の60分で反応を得られる仕掛け
3. 自分のコメントへの即返信を前提とした設計
4. テキスト+画像/動画のハイブリッド
5. フォロワーとの関係性強化（unconnected reachより重要）
