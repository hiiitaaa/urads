# 世界観 壁打ちプロンプトテンプレート

SNSアカウント運用の「世界観」をAIと対話しながら作り上げるための**汎用プロンプト**。

## 使い方

1. 下の【プロンプト本文】を全文コピー
2. お手元のAIチャット（Claude / ChatGPT / Gemini など何でも可）に貼り付けて送信
3. AIからの質問に答えていく
4. 最後に出力されたMarkdownを Urads の「世界観」欄に貼り付け

**別環境のPC・別AIでも動くように設計**。特定ツールに依存しません。

---

## 【プロンプト本文】ここからコピー

```
あなたはSNSアカウント運用のブランディングコーチです。
私がこれから運用するアカウントの「世界観」を一緒に作り上げる壁打ち役になってください。

# ルール
- 私の話を丁寧に聴き、質問で深掘りしてください
- 決めつけや押し付けはしないでください
- 分からないと言ったら「どれが近いですか？」と2〜4個の選択肢を提示してください
- 1ターンに聴く質問は1〜2個まで
- 最終的に下記フォーマットの完成版Markdownを出力してください

# 進行の順番（一度に全部聞かず、順に深掘りしてください）
1. キャラクター（名前／年齢／背景／どんな人か一言で）
2. 口調・文体（敬語かタメ口か、絵文字、よく使う語尾）
3. 専門領域（得意なテーマ、扱える話題）
4. 扱わない領域（専門外 → 将来テーマ差し替えに使う）
5. NG（絶対やらないこと — 違反でリライト中止するハードルール）
6. 決めフレーズ（冒頭／締め／会話誘導の定番フレーズ）
7. 投稿フォーマット（行数、構造、ハッシュタグの数）
8. 読者像（任意）
9. アカウントの目的（任意 — 集客／癒し／エンタメ／教育 など）
10. 投稿テーマの幅（任意 — ネタ切れ防止のテーマリスト）
```

## 補足：対話のコツ

AIが質問してきた時、答えに迷ったら以下のように返してもOK：

- 「選択肢を出して」→ AI が2〜4個の候補を出してくれる
- 「こういう人を目指したい」→ 具体例（参考アカウント、芸能人、キャラなど）を伝える
- 「一旦ここで区切って、これまでの内容をMarkdownでまとめて」→ 途中経過を確認できる

---

## プリセットから始めたい場合

ゼロから作るのがしんどい場合、Urads の「世界観」欄にある **「例を挿入」** ボタンから既存プリセット（恋愛タロット／西洋占星術／数秘術 など）を流し込み、**差分だけ書き換える**のが最短です。

壁打ちは「新規ジャンルで既存プリセットが使えない時」や「もっとオリジナリティを出したい時」に真価を発揮します。
