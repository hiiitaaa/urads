Urads 個別サーバー
セットアップガイド

自分のCloudflareアカウントでサーバーを立てる方法

所要時間: 約50分 ｜ 全21ステップ

始める前に確認

  • PC（Windows または Mac）
  • Google Chrome がインストール済み
  • Facebook アカウントがある
  • Threads アカウントがある
  • スマホ（Google Authenticator 用）
  • メモ帳アプリ（ID をメモするため）
Part 1 / 5

ツールをインストール

3つの無料ツールを入れます（5分）

1-1. Node.js をインストール

  1. ブラウザで https://nodejs.org/ を開く
  2. 緑色の 「LTS」ボタン をクリック
  3. ダウンロードしたファイルをダブルクリック
  4. 「Next」を押し続けて「Install」→ 完了
[スクリーンショット: nodejs.org の LTS ダウンロードボタン]

1-2. pnpm を有効化

PowerShell を開いて以下を入力:

corepack enable

何も表示されなければ成功です

[スクリーンショット: PowerShell で corepack enable 実行]

1-3. Git をインストール

  1. ブラウザで https://git-scm.com/download/win を開く
  2. 自動でダウンロードが始まる
  3. インストーラーを実行 → 全部デフォルトのまま「Next」
  4. 最後に「Install」→ 完了
[スクリーンショット: Git インストーラー画面]
Part 2 / 5

Cloudflare に登録

データを保存するクラウドサービスです。無料で使えます（10分）

2-1. アカウント作成

  1. ブラウザで https://dash.cloudflare.com/sign-up を開く
  2. メールアドレスとパスワードを入力
  3. 「Sign Up」をクリック
  4. メールが届く → メール内のリンクをクリック
[スクリーンショット: Cloudflare サインアップ画面]

2-2. Wrangler にログイン

PowerShell で以下を実行:

npx wrangler login
  1. ブラウザが自動で開く
  2. 「Allow」（許可）をクリック
  3. ターミナルに「Successfully logged in」と表示されれば OK
[スクリーンショット: Cloudflare Workers 認証許可画面]
Part 3 / 5

Threads API を準備

ここが一番時間がかかります。
落ち着いて1ステップずつ進めましょう（20分）

3-1. Facebook の二段階認証を ON にする

これが必須です。やらないと次に進めません！
  1. スマホで Google Authenticator アプリをインストール
  2. PC で https://www.facebook.com/settings を開く
  3. 「パスワードとセキュリティ」→「二段階認証」
  4. 「認証アプリ」を選択
  5. QR コードをスマホで読み取り → 6桁の数字を入力
[スクリーンショット: Facebook 二段階認証設定画面]

3-2. Facebook ページを作成

開発者ポータルに入るために必要です（1分で終わります）

  1. ブラウザで https://www.facebook.com/pages/create を開く
  2. 「ビジネスまたはブランド」を選択
  3. ページ名を入力（何でもOK。例:「My Tool」）
  4. カテゴリを選択（何でもOK）→「作成」
[スクリーンショット: Facebook ページ作成画面]

3-3. 開発者ポータルにアクセス

  1. ブラウザで https://developers.facebook.com/ を開く
  2. 右上の「開始する」または「マイアプリ」をクリック
  3. 開発者規約に同意
「アクセス権がありません」と出たら → 3-1 と 3-2 をやり直してください
[スクリーンショット: Meta 開発者ポータル トップ画面]

3-4. アプリを作成

  1. 「アプリを作成」をクリック
  2. 「Access Threads API」を選択（これ以外は選ばない！）
  3. アプリ名を入力（例:「Urads」）
  4. 連絡先メールアドレスを入力 →「作成」
[スクリーンショット: 「Access Threads API」選択画面]

3-5. リダイレクト URI を設定

左メニュー「Threads ログイン」→「設定」を開いて、
3つの URL を全部入力してください:

https://localhost:8890/callback
https://localhost:8890/uninstall
https://localhost:8890/delete

最後に「変更を保存」をクリック

3-6. App ID と App Secret をメモ

  1. 左メニュー「設定」→「基本」
  2. App ID の数字をコピー → メモ帳に貼り付け
  3. App Secret の「表示」をクリック → コピー → メモ帳に貼り付け
この2つは Part 4 で使います。絶対になくさないで！
[スクリーンショット: App ID と App Secret の場所]

3-7. 自分をテスターに追加

  1. 左メニュー「アプリの役割」→「役割」
  2. 「Threads テスター」の「追加」をクリック
  3. 自分の Threads ユーザー名 を入力 → 追加
[スクリーンショット: Threads テスター追加画面]

3-8. スマホで招待を承認

ここが一番迷いやすいポイントです！
PC ではなく スマホの Threads アプリ で行います。
  1. スマホで Threads アプリ を開く
  2. プロフィール → 右上の 設定アイコン
  3. 「アカウント」→「ウェブサイトのアクセス許可」
  4. 「招待」タブ → 「承認」をタップ
承認しないとこの後の認証が失敗します！
Part 4 / 5

Urads をセットアップ

もう少しで完了です！（10分）

4-1. ソースコードを取得

PowerShell で以下を1行ずつ実行:

git clone https://github.com/nohara-ce/urads.git
cd urads
[スクリーンショット: git clone 実行結果]

4-2. セットアップを実行

Windows の場合:

pnpm setup:win

Mac の場合:

pnpm setup

途中で聞かれます:

THREADS_APP_ID:→ 3-6 でメモした App ID
THREADS_APP_SECRET:→ 3-6 でメモした App Secret
最後に Workers URL が表示されます。コピーしてください！
Part 5 / 5

アプリを起動

最後のステップです！（5分）

5-1. アプリを起動する

pnpm dev

アプリウィンドウが開きます

[スクリーンショット: Urads 起動画面]

5-2. サーバーを接続

  1. セットアップ画面で 「自分のサーバーを使う」 をクリック
  2. Part 4 でコピーした Workers URL を貼り付け
  3. 「接続テスト」をクリック → 「接続OK」が出ればOK
https://urads-api.xxxxx.workers.dev
[スクリーンショット: サーバー接続設定画面]

5-3. Threads アカウントを連携

  1. 「Threads アカウントを連携する」をクリック
  2. ブラウザが開く → Threads でログイン
  3. 「許可する」をクリック
  4. アプリに戻る → 「@ユーザー名 を連携しました」
[スクリーンショット: Threads OAuth 認証画面]

セットアップ完了！

「始める」をクリックして Urads をお楽しみください

次回からの起動:

cd urads && pnpm dev

困ったときは

症状対処法
「node が見つからない」1-1 をやり直す
wrangler login でブラウザが開かない表示された URL を手動でブラウザに貼り付け
開発者ポータルで「アクセス権なし」3-1（二段階認証）と 3-2（FB ページ）を確認
OAuth 認証が失敗する3-8（スマホで招待承認）を確認
接続テストが失敗するWorkers URL が https:// で始まるか確認

ログファイル: %APPDATA%\Urads\logs\main.log
