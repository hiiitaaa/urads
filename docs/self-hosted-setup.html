✦ Part 1 / 5

ツールをインストール

3つの無料ツールを入れます

約5分で完了します

1-1. Node.js をインストール

  1. ブラウザで nodejs.org を開く
  2. 緑色の 「LTS」ボタン をクリック
  3. ダウンロードしたファイルをダブルクリック
  4. 「Next」を押し続けて「Install」→ 完了
スクリーンショット: nodejs.org のLTSダウンロードボタン

1-2. pnpm を有効化

PowerShell を開いて以下をコピー&ペースト:

corepack enable

何も表示されなければ成功です ✦

1-3. Git をインストール

  1. ブラウザで git-scm.com/download/win を開く
  2. 自動でダウンロードが始まります
  3. インストーラーを実行 → 全部「Next」でOK
  4. 最後に「Install」→ 完了
スクリーンショット: Git インストール完了画面
✦ Part 2 / 5

Cloudflare に登録

あなたのデータを安全に保存する
クラウドサービスです。無料で使えます

約10分で完了します

2-1. アカウントを作成

  1. ブラウザで dash.cloudflare.com/sign-up を開く
  2. メールアドレスとパスワードを入力
  3. 「Sign Up」をクリック
  4. 届いたメールのリンクをクリックして確認
スクリーンショット: Cloudflare サインアップ画面

2-2. Wrangler にログイン

PowerShell で以下を実行してください:

npx wrangler login
  1. ブラウザが自動で開きます
  2. 「Allow」（許可）をクリック
  3. ターミナルに「Successfully logged in」と出ればOK ✦
✦ Part 3 / 5

Threads API を準備

ここが一番時間がかかります

落ち着いて1ステップずつ進めましょう

約20分で完了します

3-1. Facebook の二段階認証をONにする

これが必須です。やらないと次に進めません！
  1. スマホで Google Authenticator をインストール
  2. PCで facebook.com/settings を開く
  3. 「パスワードとセキュリティ」→「二段階認証」
  4. 「認証アプリ」を選択 → QRコードをスマホで読み取り
  5. 表示された6桁の数字を入力 → 完了

3-2. Facebook ページを作成

開発者ポータルに入るために必要です（1分で終わります）

  1. ブラウザで facebook.com/pages/create を開く
  2. ページ名を入力（何でもOK。例:「占いラボ」）
  3. カテゴリを選択（何でもOK）→「作成」

3-3. 開発者ポータルにアクセス

  1. ブラウザで developers.facebook.com を開く
  2. 右上の「開始する」または「マイアプリ」をクリック
  3. 開発者規約に同意
「アクセス権がありません」と出たら
→ 3-1（二段階認証）と 3-2（ページ作成）を確認してください

3-4. アプリを作成

  1. 「アプリを作成」をクリック
  2. 「Access Threads API」を選択
  3. アプリ名を入力（例:「Urads」）
  4. メールアドレスを入力 →「作成」
必ず「Access Threads API」を選んでください。
他を選ぶとうまくいきません！

3-5. リダイレクト URI を設定

左メニュー「Threads ログイン」→「設定」を開いて
3つのURLを入力してください:

https://localhost:8890/callback
https://localhost:8890/uninstall
https://localhost:8890/delete

最後に「変更を保存」をクリック

3-6. App ID と App Secret をメモ

  1. 左メニュー「設定」→「基本」
  2. App ID の数字をコピー → メモ帳に貼り付け
  3. App Secret の「表示」→ コピー → メモ帳に貼り付け
✦ この2つは Part 4 で使います。なくさないで！

3-7. 自分をテスターに追加

  1. 左メニュー「アプリの役割」→「役割」
  2. 「Threads テスター」の「追加」をクリック
  3. 自分の Threads ユーザー名 を入力 → 追加
スクリーンショット: テスター追加画面

3-8. スマホで招待を承認

ここが一番迷いやすいポイントです！
PCではなくスマホのThreadsアプリで行います
  1. スマホで Threads アプリ を開く
  2. プロフィール → 右上の 設定アイコン
  3. 「アカウント」→「ウェブサイトのアクセス許可」
  4. 「招待」タブ → 「承認」をタップ
承認しないとこの後の認証が失敗します！
✦ Part 4 / 5

Urads をセットアップ

もう少しで完了です！

約10分で完了します

4-1. ソースコードを取得

PowerShell で以下を1行ずつ実行:

git clone https://github.com/nohara-ce/urads.git
cd urads

4-2. セットアップを実行

Windows:

pnpm setup:win

Mac:

pnpm setup

途中で聞かれます:

THREADS_APP_ID:→ 3-6 でメモした App ID
THREADS_APP_SECRET:→ 3-6 でメモした App Secret
✦ 最後に Workers URL が表示されます。コピーしてください！
✦ Part 5 / 5

アプリを起動

最後のステップです！

約5分で完了します

5-1. アプリを起動する

pnpm dev

アプリウィンドウが開きます ✦

スクリーンショット: Urads 起動画面

5-2. サーバーを接続

  1. セットアップ画面で「自分のサーバーを使う」をクリック
  2. Part 4 でコピーした Workers URL を貼り付け
  3. 「接続テスト」→ 「接続OK」が出ればOK ✦
https://urads-api.xxxxx.workers.dev

5-3. Threads アカウントを連携

  1. 「Threads アカウントを連携する」をクリック
  2. ブラウザが開く → Threads でログイン
  3. 「許可する」をクリック
  4. アプリに戻る →「連携しました」と表示 ✦

✦ セットアップ完了！

「始める」をクリックして
Urads をお楽しみください

次回からの起動:

cd urads && pnpm dev

☽ 困ったときは

症状対処法
「node が見つからない」1-1 をやり直す
wrangler login が開かないURLを手動でブラウザに貼り付け
開発者ポータルで「アクセス権なし」3-1 と 3-2 を確認
OAuth 認証が失敗する3-8 のスマホ承認を確認
接続テストが失敗するURLが https:// で始まるか確認
