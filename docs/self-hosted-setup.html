✦ Part 1 / 5

ツールをインストール

3つの無料ツールを入れます

約5分で完了

1-1. Node.js をインストール

  1. ブラウザで nodejs.org を開く
  2. 緑色の 「LTS」ボタン をクリック
  3. ダウンロードしたファイルをダブルクリック
  4. 「Next」→「Install」→ 完了
スクリーンショット: nodejs.org のLTSボタン

1-2. pnpm を有効化

PowerShell を開いて以下を実行:

corepack enable✦ コピー

何も表示されなければ成功です

1-3. Git をインストール

  1. ブラウザで git-scm.com/download/win を開く
  2. 自動でダウンロードが始まります
  3. インストーラーを実行 → 全部「Next」でOK
  4. 最後に「Install」→ 完了
✦ Part 2 / 5

Cloudflare に登録

データを安全に保存するクラウドサービス。無料です

約10分で完了

2-1. アカウントを作成

  1. ブラウザで dash.cloudflare.com/sign-up を開く
  2. メールアドレスとパスワードを入力
  3. 「Sign Up」をクリック
  4. 届いたメールのリンクをクリック
スクリーンショット: Cloudflare サインアップ画面

2-2. Wrangler にログイン

PowerShell で以下を実行:

npx wrangler login✦ コピー
  1. ブラウザが自動で開きます
  2. 「Allow」（許可）をクリック
  3. 「Successfully logged in」と出ればOK
✦ Part 3 / 5

Threads API を準備

ここが一番大事なパートです

落ち着いて進めましょう

約20分で完了

3-1. Facebook の二段階認証をON

⚠ これが必須です。やらないと先に進めません
  1. スマホで Google Authenticator をインストール
  2. PCで facebook.com/settings を開く
  3. 「パスワードとセキュリティ」→「二段階認証」
  4. 「認証アプリ」→ QRコードをスマホで読み取り
  5. 6桁の数字を入力 → 完了

3-2. Facebook ページを作成

開発者ポータルに入るために必要です（1分で終わります）

  1. ブラウザで facebook.com/pages/create を開く
  2. ページ名を入力（何でもOK。例:「占いラボ」）
  3. カテゴリを選択（何でもOK）→「作成」

3-3. 開発者ポータルにアクセス

  1. ブラウザで developers.facebook.com を開く
  2. 右上の「開始する」か「マイアプリ」をクリック
  3. 開発者規約に同意
「アクセス権がありません」と出たら 3-1 と 3-2 を確認

3-4. アプリを作成

  1. 「アプリを作成」をクリック
  2. 「Access Threads API」を選択
  3. アプリ名を入力（例:「Urads」）
  4. メールアドレスを入力 →「作成」
✦ 必ず「Access Threads API」を選んでください

3-5. リダイレクト URI を設定

左メニュー「Threads ログイン」→「設定」で3つ入力:

https://localhost:8890/callback✦ コピー
https://localhost:8890/uninstall✦ コピー
https://localhost:8890/delete✦ コピー

最後に「変更を保存」をクリック

3-6. App ID と App Secret をメモ

  1. 左メニュー「設定」→「基本」
  2. App ID の数字をコピー → メモ帳に
  3. App Secret の「表示」→ コピー → メモ帳に
✦ この2つは Part 4 で使います。なくさないで！

3-7. 自分をテスターに追加

  1. 左メニュー「アプリの役割」→「役割」
  2. 「Threads テスター」の「追加」をクリック
  3. 自分の Threads ユーザー名 を入力 → 追加

3-8. スマホで招待を承認

⚠ 一番迷いやすいポイント！スマホのThreadsアプリで行います
  1. スマホで Threads アプリ を開く
  2. プロフィール → 右上の 設定
  3. 「アカウント」→「ウェブサイトのアクセス許可」
  4. 「招待」タブ → 「承認」をタップ
承認しないとこの後の認証が失敗します
✦ Part 4 / 5

Urads をセットアップ

もう少しで完了です！

約10分で完了

4-1. ソースコードを取得

PowerShell で以下を1行ずつ実行:

git clone https://github.com/YOUR_USERNAME/urads.git✦ コピー
cd urads✦ コピー

4-2. セットアップを実行

Windows:

pnpm setup:win✦ コピー

Mac:

pnpm setup✦ コピー

途中で聞かれます:

THREADS_APP_ID→ 3-6 の App ID
THREADS_APP_SECRET→ 3-6 の App Secret
✦ 最後に Workers URL が表示されます。コピー！
✦ Part 5 / 5

アプリを起動

最後のステップです！

約5分で完了

5-1. アプリを起動する

pnpm dev✦ コピー

アプリウィンドウが開きます

スクリーンショット: Urads 起動画面

5-2. サーバーを接続

  1. セットアップ画面で「自分のサーバーを使う」をクリック
  2. Part 4 でコピーした Workers URL を貼り付け
  3. 「接続テスト」→「接続OK」が出ればOK
https://urads-api.xxxxx.workers.dev

5-3. Threads アカウントを連携

  1. 「Threads アカウントを連携する」をクリック
  2. ブラウザが開く → Threads でログイン
  3. 「許可する」をクリック
  4. アプリに戻る →「連携しました」と表示

✦ セットアップ完了！

「始める」をクリックして
Urads をお楽しみください

cd urads && pnpm dev✦ コピー

次回からはこのコマンドだけでOK
