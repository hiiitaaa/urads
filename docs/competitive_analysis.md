# Urads 競合分析（Threads自動投稿ツール市場）

## 市場概況

- GAS（Google Apps Script）+ Threads公式API の買い切りツールが主流
- 販売プラットフォーム: Brain, note.com, BOOTH, Coconala
- 価格帯: 2,980〜24,800円（買い切り）
- Google スプレッドシートで投稿管理が標準

## 主要競合

| ツール | 価格 | 特徴 |
|--------|------|------|
| GASラボ | 2,980円/ツール | 市場リーダー。ツール種類豊富、パッチ対応早い |
| SWELLER | 2,980円 | 動画投稿対応（珍しい）。1%タイムアウト問題あり |
| ThreadsCAT | 6,980円 | Brain販売。ドキュメント分かりにくいとの評価 |
| しんたろ。 | 24,800円 | 高価格帯。インサイト取得対応 |
| SANANEBLOG | 無料+有料 | フリーミアム。キーワード自動返信対応 |
| Insane Projects | 不明 | 大学生が開発して話題に |
| Coconala各種 | ~10,000円 | カスタム開発サービス |

## Urads vs GAS系ツール 完全比較

| 機能 | GAS系ツール | Urads | 優位 |
|------|-----------|-------|------|
| 予約投稿数 | **20件/アカウント**（GASトリガー制限） | **無制限**（D1 + Cron） | Urads |
| 投稿タイミング精度 | 1-15分のズレあり | **秒単位で正確** | Urads |
| トークン更新 | **手動**（60日で切れる） | **自動リフレッシュ**（Cron） | Urads |
| 動画投稿 | ほぼ未対応 | **対応予定** | Urads |
| 画像投稿 | URL指定のみ（ローカルファイル不可） | **ドラッグ&ドロップ**（R2経由） | Urads |
| カルーセル | 一部対応（6分制限で不安定） | **安定対応**（時間制限なし） | Urads |
| リプライ自動化 | ほぼ未対応（GASでWebhook困難） | **Webhook + ルールエンジン** | Urads |
| AI文章生成 | なし | **Claude API / ollama** | Urads |
| 分析ダッシュボード | スプレッドシートのみ | **専用UI + グラフ** | Urads |
| 複数アカウント | 一部対応 | **最大5アカウント + 切替** | Urads |
| リサーチ（競合分析） | なし | **ベンチマーク + バズ検知** | Urads |
| 画像エディタ | なし | **Fabric.js テンプレエディタ** | Urads |
| エラー検知 | 投稿失敗に気付かない | **リアルタイム通知** | Urads |
| BAN対策 | なし/自己責任 | **安全レイヤー内蔵** | Urads |
| API変更への対応 | 開発者次第（2025/4月に大量死亡） | **electron-updater で自動更新** | Urads |
| UI | スプレッドシート | **専用デスクトップアプリ** | Urads |
| セットアップ | Meta App作成 + GAS設定（難しい） | **アプリ起動 → OAuth認証のみ** | Urads |
| 月額費用 | 0円（GAS無料） | 0円（Cloudflare無料枠） | 引き分け |
| PCオフ時 | 動く（GASはGoogle側で実行） | **動く**（Cloudflare Cron） | 引き分け |

## GAS系の致命的弱点（Uradsで全部解決）

1. **予約20件制限** → Urads: 無制限
2. **トークン手動更新** → Urads: 自動リフレッシュ
3. **投稿タイミングのズレ** → Urads: 秒単位精度
4. **エラーに気付かない** → Urads: リアルタイム通知
5. **API変更で動かなくなる** → Urads: electron-updater で自動更新
6. **スプシUIが使いにくい** → Urads: 専用アプリ

## Urads独自機能（競合に一切ない）

- AI文章生成（占いプリセット・カスタムプロンプト）
- リプライ自動化（Webhook + キーワードマッチ / ランダム返信）
- 競合リサーチ + バズ検知
- 画像/動画テンプレートエディタ
- BAN防止安全レイヤー（投稿間隔・クォータ管理・ジッター）
- 投稿プレビュー

## 価格戦略の方向性

- GAS系が2,980〜24,800円の買い切り → Uradsは機能差で月額制が正当化できる
- 参考: 月額980〜2,980円 or 年額プラン
- 無料トライアル（1アカウント・基本機能のみ）で導入ハードルを下げる

## ユーザーがGAS系で不満に感じているポイント（レビューから）

1. トークン期限切れで投稿が止まる（最多の不満）
2. セットアップが難しい（Meta App作成が壁）
3. API変更で突然動かなくなる（2025年4月の大量死亡事件）
4. 画像投稿がURL指定のみで面倒
5. 投稿失敗に気付けない
6. ドキュメントが分かりにくい
7. アカウント情報をスプシに入力する不安
